# 人才市场

随着 `GoFrame` 社区发展，越来越多的企业和开发人员希望通过 `GoFrame` 社区找到合适的工作机会和人才资源。为此，我们专门创建了这个仓库。

有谁在 [招聘](https://github.com/gogf/jobs/issues?q=state%3Aopen%20label%3A%E6%8B%9B%E8%81%98) / [求职](https://github.com/gogf/jobs/issues?q=state%3Aopen%20label%3A%E6%B1%82%E8%81%8C) ？

我要发布 [招聘](https://github.com/gogf/jobs/issues/new?template=0-hiring.yml) / [求职](https://github.com/gogf/jobs/issues/new?template=1-seeking.yml) 信息！

## 📋 仓库说明

这是一个专门为**开发人员求职和企业招聘**服务的仓库。通过 GitHub Issue 的形式发布招聘与求职信息。

## 🤝 参与方式

欢迎所有开发人员和企业参与！提交 Issue 时请遵循社区规范，确保信息准确、合法。

---

如有任何问题或建议，欢迎通过 Issue 反馈！
